# Code of conduct

SaltanatbotV2 welcomes contributors of every background and experience level. Project spaces must remain safe, technical and respectful.

Expected behavior:

- discuss ideas and code without attacking people;
- give actionable feedback and assume good faith while evidence is incomplete;
- respect privacy, credentials and coordinated vulnerability disclosure;
- disclose conflicts of interest and avoid misleading performance claims;
- accept maintainer decisions on safety, scope and moderation.

Harassment, discrimination, threats, doxxing, sexualized conduct, spam, market manipulation and publishing another person's secrets are not accepted.

Report conduct concerns privately to project maintainers through repository moderation or security channels. Maintainers may edit or remove content, issue a warning, temporarily restrict participation or permanently ban an account. Reports should be handled confidentially and without retaliation.
