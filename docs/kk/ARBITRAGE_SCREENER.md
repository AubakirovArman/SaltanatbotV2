# Биржааралық арбитраж скринері

**Скринер** бөлімі API кілттерінсіз Binance және Bybit биржаларындағы ортақ USDT жұптарын тексереді.
Әр жұп үшін екі бағыт есептеледі:

1. Binance spot-ты ask бағасымен сатып алып, Bybit linear perpetual-ды bid бағасымен short ашу;
2. Bybit spot-ты ask бағасымен сатып алып, Binance USD-M perpetual-ды bid бағасымен short ашу.

Кестеде жалпы spread, реттелетін шығын бағасы, таза айырма, екі үздік бағада қолжетімді көлем және
funding мөлшерлемесі көрсетіледі.
Абсолюттік айырмасы 20%-дан жоғары жолдар қабылданбайды: бұл орындалатын мүмкіндікке қарағанда
бірдей ticker бар әртүрлі актив, redenomination немесе ескірген нарық болуы ықтимал.

```text
жалпы spread (bp) = (perpetual bid - spot ask) / spot ask × 10 000
таза айырма (bp)  = жалпы spread - болжамды толық шығын
қолжетімді көлем  = min(spot ask × ask size, perpetual bid × bid size)
```

Шығын өрісіне кіру/шығу комиссиялары мен slippage қорын енгізу керек. Funding ұстау мерзімі мен
болашақ мөлшерлемелер белгісіз болғандықтан таза айырмаға автоматты түрде қосылмайды.

API деректері қатар оқылады және серверде екі секунд кэштеледі. Барлық көз уақытша істемесе,
30 секундтан ескі емес соңғы snapshot ескерту белгісімен көрсетілуі мүмкін. Жасырын browser tab
автоматты polling жасамайды.

Бұл тек зерттеу құралы және order жібермейді. Екі биржаның бағалары атомарлы емес. Комиссия,
нарық тереңдігі, funding, қарыз қолжетімділігі, аударым, latency және liquidation тәуекелі көрінген
айырманы толық жоюы мүмкін. Оң нәтиже пайдаға кепілдік бермейді.
