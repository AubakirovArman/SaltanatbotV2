# OKX ашық REST адаптері

Күйі: read-only backend компоненті 2026 жылғы 14 шілдеде тексерілді,
`/api/market-data/okx/*` арқылы қолжетімді және normalized `/api/instruments` registry құрамына
кіреді. Ол live arbitrage scanner, chart UI және private trading-ке қосылған жоқ.

API кілттерін қолданбайтын адаптер OKX API v5 ашық деректерін қолдайды:

- `SPOT`, мерзімсіз `SWAP` және мерзімді `FUTURES` құралдарының метадеректері;
- тикерлер тізімі және бір құралдың top-of-book мәндері;
- 1–400 деңгейлік толық REST order-book snapshot;
- `SWAP` үшін ағымдағы funding бағасы, кестесі және тарихы.

Биржаның `instId` мәні дефистерін жоғалтпай сақталады. Spot мөлшері базалық активпен өлшенеді.
Деривативтерде lot қадамы мен стакан мөлшері контрактпен өлшенеді; `ctVal`, `ctMult`, `ctValCcy`,
linear/inverse түрі, есеп айырысу валютасы және expiry уақыты бөлек сақталады. Ашық метадеректер
әр өнім үшін әмбебап minimum notional бермейді, сондықтан `minimumNotional: 0` шектеу жоқ дегенді
емес, мән белгісіз екенін білдіреді.

Funding аралығы тұрақты сегіз сағат деп қабылданбайды: ол `fundingTime` және `nextFundingTime`
бойынша есептеледі. Ағымдағы funding сұрауы сәтсіз болса, нәтиже қайтарылмайды. Тек тарих
қолжетімсіз болса, ағымдағы баға дереккөз қатесімен бірге сақталады.

Әр сұраудың timeout және жауап көлемі шектелген, caller cancellation қолдауы бар. HTTP күйі, OKX
`code`, JSON және мағыналық инварианттар тексеріледі. Қиылысқан top-of-book, реттелмеген немесе
қиылысқан стакан, жарамсыз сан, timestamp не sequence қауіпсіз түрде қабылданбайды. Registry OKX
SPOT/SWAP/FUTURES үшін соңғы сәтті snapshot-ты бөлек сақтайды, сондықтан бір дереккөздің уақытша
ақауы бұрын тексерілген құралдарды өшірмейді.

Әзірге WebSocket және стаканды қалпына келтіру, scanner/UI интеграциясы, margin/options/spreads,
аккаунтты оқу, қарыз алу, аударымдар және ордер орындау әдейі қосылған жоқ. Capability manifest
жеке орындауды `false` деп нақты көрсетеді.

Толық техникалық сипаттама: [OKX public adapter](../OKX_PUBLIC_ADAPTER.md). Ресми дереккөз:
[OKX API v5](https://www.okx.com/docs-v5/en/).
