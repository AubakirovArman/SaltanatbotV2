27 757 жол қосылды және 12 039 жол жойылды. Бұл — ерте alpha, нақты қаражатпен саудаға дайындық

## Негізгі өзгерістер

- Pine Script v4–v6 импорт модульдік compiler package-ке бөлінді: parser, semantic analysis,
  symbol scopes, typed diagnostics және fail-closed conversion.
- Pine қолдауы multiline state, `fill()`, plots, inputs, alerts, drawings, tuples, switches,
  user functions және қосымша numeric/boolean өрнектермен кеңейді.
- Strategy Studio таңдалған Binance/Bybit дереккөзін және `request.security` series-ті қолданады.
- Ортақ `strategy-core` және `backtest-core` preview/backtest/paper/live parity-ді күшейтті.
- Backtest data provenance, deterministic traces және signal/fill/state explanations алды.
- Графикте keyboard және screen reader үшін OHLC, signals және trades HTML кестелері бар.
- Live engine durable order lifecycle, polling, Binance/Bybit private streams және restart
  reconciliation алды; белгісіз outcome automation-ды pause етеді.
- Chart, Strategy Studio, optimizer және Trading workspace ағылшын/орыс тілдеріне локализацияланды.
- Browser E2E, Pine fuzz, parity/golden және exchange recovery тесттері қосылды.
- Generated references, docs CI, testnet smoke, SBOM, SHA-256 және Sigstore attestations қосылды.

Толық тізім: [CHANGELOG](../../CHANGELOG.md). Құжат мәртебесі: [DOCUMENTATION_STATUS](../DOCUMENTATION_STATUS.md).

## Тексеру

```bash
git clone https://github.com/AubakirovArman/SaltanatbotV2.git
cd SaltanatbotV2
npm install
npm run check
npm run lint
npm test
npm run build
npm run dev
```

Алдымен backtest және paper режимін қолданыңыз. Pine warnings TradingView-пен бірдей candles арқылы
қолмен салыстырылуы тиіс. Live trading эксперименттік күйде.
