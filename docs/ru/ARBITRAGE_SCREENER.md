# Межбиржевой арбитражный скринер

Раздел **Скринер** без API-ключей проверяет общие USDT-пары Binance и Bybit. Для каждой пары
рассчитываются два направления:

1. купить spot Binance по ask и открыть short бессрочного linear-фьючерса Bybit по bid;
2. купить spot Bybit по ask и открыть short бессрочного USD-M-фьючерса Binance по bid.

Скринер показывает грязный спред, настраиваемую оценку общих издержек, чистый спред, доступный объём
в лучших ценах и funding. Объём ограничен меньшей из двух сторон.
Разница больше 20% по модулю отбрасывается: это чаще коллизия одинаковых тикеров разных активов,
реденоминация или устаревший рынок, а не исполнимая возможность.

```text
грязный спред (bp) = (bid фьючерса - ask спота) / ask спота × 10 000
чистый спред (bp)  = грязный спред - оценка общих издержек
доступный объём    = min(ask спота × размер ask, bid фьючерса × размер bid)
```

В общие издержки следует заложить комиссии входа и выхода и запас на проскальзывание. Funding
показывается отдельно: заранее неизвестны срок удержания и будущие ставки.

API опрашиваются параллельно, результаты кэшируются на сервере на две секунды. Временный полный
отказ может показать последний успешный снимок не старше 30 секунд с явной пометкой. В скрытой
вкладке браузер прекращает автоопрос.

Скринер работает только для исследования и не отправляет ордера. Котировки двух бирж не атомарны.
Комиссии, реальная глубина, funding, доступность займа, переводы, задержка и риск ликвидации могут
полностью убрать найденную разницу. Положительный результат не является гарантией прибыли.
