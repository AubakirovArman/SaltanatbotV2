# Обновление SaltanatbotV2: 90 коммитов за 10–11 июля 2026

Срез включает ровно 90 коммитов (`d5c45c6`…`b6ca124`): 363 изменённых файла, 27 757 добавленных
и 12 039 удалённых строк. Это development snapshot ранней alpha-версии, а не обещание готовности
к торговле реальными средствами.

## Главное

- Pine Script v4–v6 импорт вынесен в отдельный модульный compiler package: parser, semantic analysis,
  symbol scopes, typed diagnostics, expression/statement/drawing lowering и fail-closed ошибки.
- Поддержка Pine расширена для multiline object state, `fill()`, plots, inputs, alerts, drawings,
  tuples, switches, user functions и дополнительных numeric/boolean конструкций.
- Strategy Studio корректно использует выбранную Binance/Bybit и загружает `request.security` series.
- Созданы общие `strategy-core` и `backtest-core`; preview, backtest, paper и live используют
  согласованные evaluator/TA-контракты.
- Backtest получил data provenance, детерминированные execution traces и объяснения сигналов,
  исполнений, позиции и equity.
- График получил доступные HTML-таблицы OHLC, сигналов и сделок для клавиатуры и screen reader.
- Live engine получил durable lifecycle ордеров, polling, private streams Binance/Bybit,
  idempotent event ingest и startup reconciliation.
- Защитные SL/TP должны быть подтверждены биржей; неоднозначный исход ставит автоматизацию на паузу.
- Интерфейс графика, Strategy Studio, optimizer и Trading workspace локализован на английский/русский.
- Добавлены browser E2E, parser fuzz, parity/golden, exchange failure/recovery тесты.
- Добавлены generated API/block/Pine references, docs CI, testnet smoke, SBOM, checksums и Sigstore.

Подробная техническая история находится в [CHANGELOG](../../CHANGELOG.md), а текущее состояние — в
[реестре документации](../DOCUMENTATION_STATUS.md).

## Как проверить

```bash
git clone https://github.com/AubakirovArman/SaltanatbotV2.git
cd SaltanatbotV2
npm install
npm run check
npm run lint
npm test
npm run build
npm run dev
```

Сначала проверяйте стратегии в backtest и paper. Pine-предупреждения требуют ручного сравнения с
TradingView на одинаковых свечах. Live trading остаётся экспериментальным.
