# Безопасность и сообщение об уязвимостях

SaltanatbotV2 находится в alpha-стадии. Исправления безопасности выпускаются для актуальной ветки `main`.

Об уязвимости сообщайте через приватный **Security advisory** репозитория. Не публикуйте exploit, API-ключи, access token, `.env`, `backend/data/`, SQLite-файлы или реальные данные счёта в открытых issues. Если приватный канал недоступен, создайте issue только с просьбой предоставить безопасный канал связи.

Для воспроизведения используйте `DEMO_MODE=1`, paper mode или testnet. Биржевые ключи должны быть без права вывода средств и, по возможности, ограничены по IP. Любой секрет, попавший в лог, скриншот или commit, необходимо немедленно заменить.

Каноническая полная политика: [SECURITY.md](../../SECURITY.md).
