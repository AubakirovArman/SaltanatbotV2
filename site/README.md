# GitHub Pages source

This directory contains the dependency-free multilingual public project page.

- `index.html`: English
- `ru/index.html`: Russian
- `kk/index.html`: Kazakh
- `styles.css`: shared accessible responsive presentation

`.github/workflows/pages.yml` uploads this directory without a build step. Keep all asset links
relative so preview and the `/SaltanatbotV2/` project subpath behave identically.
